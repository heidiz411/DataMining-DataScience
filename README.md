# ITC4103 เหมืองข้อมูลและเทคนิคการพยากรณ์สำหรับวิทยาการข้อมูล (Data Mining and Forecasting Techniques for Data Science)

## 1. จุดมุ่งหมายของรายวิชา
รายวิชานี้มีจุดมุ่งหมายเพื่อให้นักศึกษามีความรู้ความเข้าใจในหลักการ ข้อมูลขุดค้น (Data Mining) กระบวนการ และเทคนิคสำคัญ เช่น การวิเคราะห์ตะกร้าตลาดข้อมูล การแบ่งประเภทข้อมูล การแบ่งกลุ่มข้อมูล และเทคนิคการพยากรณ์ข้อมูล รวมถึงสามารถประยุกต์ใช้ภาษาโปรแกรม Python และเครื่องมือทางสถิติในการวิเคราะห์และสร้างโมเดลการขุดค้นข้อมูลและการพยากรณ์ได้อย่างเหมาะสม นอกจากนี้ยังเน้นการประยุกต์ใช้งานจริงในภาคอุตสาหกรรมและธุรกิจ

## 2. Learning Outcomes
เมื่อเรียนจบรายวิชานี้ นักศึกษาจะสามารถ:
- อธิบายหลักการและกระบวนการขุดค้นข้อมูล เทคนิคการพยากรณ์ และการประยุกต์ใช้วิทยาการข้อมูลได้ (CLO1)
- ประยุกต์ใช้เทคนิคการขุดค้นข้อมูล เช่น การวิเคราะห์ตะกร้าตลาด การแบ่งประเภท และการแบ่งกลุ่มข้อมูลในการแก้ปัญหาได้ (CLO2)
- ใช้ภาษา Python และเครื่องมือทางสถิติในการวิเคราะห์ข้อมูล สร้างมโนภาพ และทำการพยากรณ์ได้ (CLO3)
- ประเมินและเปรียบเทียบประสิทธิภาพของโมเดลการขุดค้นข้อมูลและการพยากรณ์ได้อย่างเหมาะสม (CLO4)
- นำเสนอผลงานและทำงานร่วมกับผู้อื่นได้อย่างมีประสิทธิภาพ (CLO5)

## 3. วิธีการจัดการเรียนการสอน
- การบรรยายและอธิบายทฤษฎีควบคู่กับการสาธิตโค้ด Python เพื่อให้นักศึกษาเข้าใจหลักการและการประยุกต์ใช้งานจริง
- การปฏิบัติการเขียนโปรแกรมในห้องปฏิบัติการ เพื่อฝึกทักษะเขียนโค้ดและวิเคราะห์ข้อมูลด้วยเครื่องมือจริง
- Workshopping และ Project-Based Learning เพื่อให้นักศึกษาประยุกต์ใช้ความรู้ในกรณีศึกษาและโครงงาน
- การอภิปรายและแลกเปลี่ยนความคิดเห็นในชั้นเรียน
- แบบทดสอบย่อยและการประเมินผลงานนักศึกษาอย่างต่อเนื่อง

## 4. หัวข้อที่ครอบคลุมในรายวิชา
- หลักการและกระบวนการขุดค้นข้อมูล (Data Mining Principles and Processes)
- การเตรียมและทำความสะอาดข้อมูล (Data Preprocessing)
- เทคนิคการวิเคราะห์ตะกร้าตลาดและกฎความสัมพันธ์ (Market Basket Analysis and Association Rules)
- การแบ่งประเภทข้อมูล (Classification Techniques)
- การแบ่งกลุ่มข้อมูล (Clustering Techniques)
- เทคนิคการพยากรณ์ (Forecasting Techniques)
- การใช้ภาษา Python และไลบรารีสำหรับ Data Science
- กรณีศึกษาและการประยุกต์ใช้งานจริงในอุตสาหกรรมและธุรกิจ

| สัปดาห์ | หัวข้อ | กิจกรรมการเรียนการสอน |
